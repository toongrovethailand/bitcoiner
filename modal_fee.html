💸 ค่าธรรมเนียม (Fee) & Block Template

ทำไมต้องจ่ายค่าธรรมเนียม (Tx Fee)?

พื้นที่จัดเก็บข้อมูลในหนึ่งบล็อก (Block Space) มีจำกัดแค่ประมาณ 4,000,000 Weight Units (WU) หรือเทียบเท่า 4MB ดังนั้นเมื่อมีคนต้องการส่งธุรกรรมเยอะกว่าที่บล็อกจะรับไหว ผู้คนจึงต้อง จ่ายค่าผ่านทางประมูลแย่งพื้นที่กัน เพื่อสร้างแรงจูงใจให้นักขุดหยิบธุรกรรมของตนลงบล็อกเป็นคิวแรกๆ

การพิจารณาความคุ้มค่า (sat/vB)

ในระบบของบิตคอยน์ การส่งเงิน 10 บาท หรือ 100 ล้านบาท ไม่ส่งผลต่อค่าธรรมเนียม สิ่งที่นักขุดพิจารณาคือ "ขนาดข้อมูล" (Data Size) ของธุรกรรมนั้นๆ

นักขุดจะนำค่า Fee ที่ผู้ใช้ยอมจ่าย มาหารด้วยขนาดของธุรกรรม (vB) ออกมาเป็นค่า sat/vB จากนั้นนักขุดจะเลือกกวาดธุรกรรมที่ให้ค่าความหนาแน่นของ sat/vB เยอะที่สุดใส่บล็อกให้เต็ม เพื่อรีดรายได้ให้คุ้มค่าที่สุด

Block Template (การทำงานในโลกจริง)

ในระบบจริง นักขุดที่คุมเครื่อง ASIC ไม่ได้เป็นคนมานั่งเลือกธุรกรรมเองเหมือนใน Simulator นี้!

  • ผู้ที่ทำหน้าที่คัดกรองธุรกรรมจาก Mempool คือ Mining Pool (กลุ่มที่เปิดให้บริการรวบรวมพลังขุด) พวกเขาจะรัน Full Node และสร้างสิ่งที่เรียกว่า Block Template ที่อัดแน่นไปด้วยธุรกรรมที่ให้กำไรสูงสุด
  • จากนั้น Pool จะทำการย่อยข้อมูลเหล่านั้นจนเหลือแค่ Block Header (ขนาด 80 bytes) แล้วส่งไปมอบหมายให้เครื่องขุด ASIC ทั่วโลกช่วยกันสุ่มหา Nonce
  • ดังนั้น เครื่อง ASIC จึง "ตาบอด" พวกมันไม่รู้ด้วยซ้ำว่ากำลังประมวลผลธุรกรรมอะไร หรือโอนให้ใคร หน้าที่เดียวของมันคือการสับตัวเลขผ่านฟังก์ชัน SHA-256d แบบไม่ลืมหูลืมตาเท่านั้น
