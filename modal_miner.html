⛏️ Node Miner

Node Miner คืออะไร มีหน้าที่อะไร?

Node Miner (นักขุด) คือผู้ให้บริการพลังการประมวลผลแก่เครือข่าย หน้าที่หลักคือการนำธุรกรรมที่รออยู่ใน Mempool มามัดรวมกันเป็นบล็อก และทุ่มเทกำลังขุดหาตัวเลข (Nonce) เพื่อแก้สมการทางคณิตศาสตร์ที่เรียกว่า Proof-of-Work (PoW) หากทำสำเร็จเป็นคนแรก จะได้รับสิทธิ์ในการเพิ่มบล็อกใหม่ลงในบล็อกเชนและรับรางวัล

การทำงานในอดีต (ยุคซาโตชิ) แตกต่างกับปัจจุบันอย่างไร?

ในยุคแรกเริ่ม (ปี 2009) ทุกคนสามารถเป็นนักขุดได้ด้วย CPU คอมพิวเตอร์ธรรมดาในบ้าน (Solo Mining) โปรแกรม Bitcoin Core สมัยนั้นทำหน้าที่ทั้งการเป็นกระเป๋าเงิน, ตรวจสอบเครือข่าย, และขุดไปพร้อมๆ กันในตัวเดียว

แต่ในปัจจุบัน ความยากในการแก้สมการเพิ่มขึ้นมหาศาล การขุดจึงกลายเป็น อุตสาหกรรมขนาดใหญ่ ต้องใช้เครื่องขุดเฉพาะทางที่เรียกว่า ASIC ซึ่งประมวลผล SHA-256d ได้ล้านล้านครั้งต่อวินาที นอกจากนี้ นักขุดยังต้องรวมตัวกันเป็น Mining Pool เพื่อแบ่งปันกำลังขุดและผลกำไรร่วมกัน

ปัจจุบันทำงานยังไง?

  • เครื่อง ASIC ในปัจจุบันทำหน้าที่เป็นแค่ "เครื่องจักรคิดเลขตาบอด" มันไม่รู้ด้วยซ้ำว่ากำลังประมวลผลธุรกรรมอะไร
  • สิ่งที่จัดแจงธุรกรรมและสร้าง Block Template คือเซิร์ฟเวอร์ของ Mining Pool
  • เครื่อง ASIC จะรับแค่ส่วนหัวของบล็อก (Block Header) มาทำการสุ่มหาค่า Nonce ไปเรื่อยๆ จนกว่าจะได้ค่า Hash ที่น้อยกว่า Target ตามที่เครือข่ายกำหนด
