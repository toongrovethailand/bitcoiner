# bitcoiner
"Miner เป็นเพียงคนงาน ส่วน Validator คือผู้รักษากติกาโลก"  โปรเจกต์นี้ได้รับแรงบันดาลใจจากแนวคิด Right Shift เพื่อสร้างเครื่องมือการเรียนรู้ (Simulator) ที่จะพาคุณไปสัมผัสหัวใจของ Bitcoin Node ตั้งแต่การคัดเลือกธุรกรรมจาก Mempool การทำ Proof of Work ไปจนถึงการกระจายข้อมูลผ่าน P2P Network ทั่วโลก เพื่อพิสูจน์ว่าทำไม "กติกา" ถึงสำคัญกว่า "แรงขุด"
